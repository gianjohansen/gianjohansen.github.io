WEB APPLICATION
Camperdown Program Online
An online component for the speech therapy program from the Australian Stuttering Research Centre. Designed to supplement the in-person clinic sessions that their patients attend.

Our client for this project was the ASRC.

Web Development, Web Design, UI/UX Design, HTML5, CSS3, Javascript, jQuery, Python, Django, Bootstrap, Git, UML.
My Contributions
  • I was part of the frontend team during the development and release of the service.
  • I prototyped the UX design and aesthetic, first by hand, then in Photoshop and Balsamiq.
  • I created UML diagrams during the design stage and worked from them during implementation.
  • I did interface implementation work in Django using Python and HTML/CSS/jQuery.
  • I used the web frameworks Bootstrap and Skeleton to scaffold and code the site.
  • I worked as part of a team using Git for source control and Redmine for project management.

WEB APPLICATION
Touchstone
A web application implementation of the database benchmarking specification TPC-C. It allows users to test simulated transactions against a relational database to evaluate its performance.

This was a 12-month project done with the CSIRO under senior researcher Dr Shiping Chen.

Web Development, Web Design, UI/UX Design, HTML5, CSS3, Javascript, jQuery, PHP, MySQL, Bootstrap, Business Analysis, Project Management, Agile Development, Extreme Programming.
My Contributions
  • I completed both the frontend and backend development under the supervision of the CSIRO.
  • I did interface implementation work with HTML/CSS/JQuery and Bootstrap.
  • I prototyped the website design, first on paper and then in Photoshop.
  • I prototyped the UX design using Balsamiq.
  • I evaluated and modified the UX using live-user testing and A/B testing.
  • I created UML diagrams during the design stage.
  • I attended client consultations to assist in our business analysis work.
